# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report them privately via the contact form on [igorclauss.de](https://igorclauss.de) or by opening a GitHub Security Advisory on the repository if you have access.

You can expect an initial response within 72 hours.
