# Contributing

Thank you for considering a contribution to **Advanced Table Export for Filament**!

## Development setup

1. Fork and clone the repository
2. Install dependencies:

```bash
composer install
```

3. Run the code style fixer:

```bash
vendor/bin/pint
```

## Pull requests

- Keep changes focused — one feature or fix per PR
- Match existing code style and naming conventions
- Update [CHANGELOG.md](CHANGELOG.md) under `[Unreleased]` for user-facing changes
- Add or update tests when behavior changes (test suite expands with v0.2.0)

## Reporting issues

Use the [GitHub issue tracker](https://github.com/OccTherapist/advanced-table-export-for-filament/issues) and include:

- PHP, Laravel, and Filament versions
- Steps to reproduce
- Expected vs. actual behavior

## Code of conduct

Be respectful and constructive. We're all here to build useful tools for the Filament community.
